# YT Console Downloader

I'm tired looking for services to download my favorite music from YouTube.

So I decided to create a simple console utility.

## Description

There are 3 options:

1. Download MP4
2. Download MP3
3. Settings

If you need more features, you
may [DM me](https://github.com/leo-proger/yt-console-downloader?tab=readme-ov-file#contacts)

## Prerequisites

- FFmpeg ([installation guide](https://github.com/oop7/ffmpeg-install-guide?tab=readme-ov-file))

## Usage

Download executable file from the latest [Release](https://github.com/leo-proger/yt-console-downloader/releases).
